# Práctica 1er Parcial - Calculadora Matemática
**Estudiante:** Starlyn Eduardo Escalante
**Matrícula:** 2025-1089

## Descripción
Este programa es una calculadora básica desarrollada en Java que aplica los fundamentos de la Programación Orientada a Objetos (POO). 

### Funcionalidades:
- Suma, resta, multiplicación y división de dos números.
- Validación con `if` para evitar errores al dividir por cero.
- Menú interactivo que se repite hasta que el usuario decide salir.
- Uso de `Scanner` para la entrada de datos por consola.
